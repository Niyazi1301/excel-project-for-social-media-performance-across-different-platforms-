
# Social Media Performance Analysis (Excel Project)

## 📊 Project Overview
This project analyzes social media performance across different platforms using Excel. It focuses on engagement metrics, campaign effectiveness, and customer retention.

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning Techniques

## 📈 Key Analysis
- Data cleaning and preprocessing
- Engagement analysis
- Platform performance comparison
- Campaign effectiveness
- Customer retention insights

## 📂 Files Included
- Excel analysis files for each task

## 📌 Conclusion
This project helps in understanding performance trends across platforms and supports better marketing decisions.
